    Portals

    October 1, 2020
    Category Experiments

    Portals are a new feature of the web similar to iframes with more emphasis on speed and user experience. The portal element is only available on Chromium-based browsers under a preference flag. The corresponding specification is still under active discussion.

    Unfortunately, some research over this new feature found critical issues, including new XS-Leaks 1.

    ID Leaks #

    Portals can be abused as an alternative for the ID Attribute XS-Leak. If the website sets framing protections, the same technique can be applied using the portal element instead 2.
